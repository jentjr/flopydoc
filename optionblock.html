flopy.utils.optionblock Module¶

class flopy.utils.optionblock.OptionBlock(options_line, package, block=True)¶

Parent class to for option blocks within Modflow-nwt models. This class contains base information and routines that can be shared throughout all option block classes.

Parameters:
options_line : str

single line based options string

package : flopy.pakbase.Package instance

valid packages include ModflowWel, ModflowSfr2, ModflowUzf1

block : bool

flag to write as single line or block type
